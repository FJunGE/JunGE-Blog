## Junge Blog

使用laravel5.7，Mysql数据库，ubuntu16.04进行开发，开发一款简易博客，博客功能涵盖了以下功能点

- 用户个人中心
- 注册/登录
- 发表博客
- 用户关注
- 安全验证
- 发布邮件系统


## 知识点概要

- 授权策略 Policies
- 数据迁移 Migrate
- 数据填充 Seeder
- 模型工厂 Factory
- 路由 Route
- 中间件 Middleware
- 事件 Event
- 邮件系统 Email
